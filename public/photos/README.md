# Photos

Drop your `.jpg / .jpeg / .png / .webp / .avif` files into this folder.

The gallery page (`/zh/gallery` and `/en/gallery`) automatically picks up everything in this directory and renders it in a grid. No code changes needed.

Tips:
- Keep file sizes reasonable (under ~2 MB each) for fast loading.
- Use square or landscape photos — they fit the grid best.
- File names are used as alt text by default; rename for better SEO if you care.
